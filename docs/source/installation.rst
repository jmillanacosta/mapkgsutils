##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/mapkgsutils>`_ with uv:

.. code-block:: console

    $ uv pip install mapkgsutils

or with pip:

.. code-block:: console

    $ python3 -m pip install mapkgsutils

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com//mapkgsutils.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com//mapkgsutils.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com//mapkgsutils.git
    $ cd mapkgsutils
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
